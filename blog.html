Question-1: What are the different ways to select an element in the DOM?

1. Selecting by ID - getElementById()

2. Selecting by ClassName - getElementsByClassName()

3. Selecting by TagName - getElementsByTagName()

4. With querySelector - querySelector()

5. With querySelectorAll - querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

1.innerHTML -> এটি দিয়ে একটি Element এর ভিতরের সকল HTML Tags পাওয়া যায় এবং নতুন HTML Tags সেট করা যায় এবং তার ভিতর থাকা Comments গুলোও।

2.innerText -> এটি দিয়ে একটি Element এর ভিতরের সকল Text পাওয়া যায় এবং নতুন Text সেট করা যায় এবং এটি HTML Tags দেখায় না। এটি CSS স্টাইলিং এর দিকে লক্ষ রাখে, তাই যদি টেক্সট লুকানো থাকে এটি দেখায় না।

3.textContent -> এটি দিয়ে একটি Element এর ভিতরের সকল Text পাওয়া যায় এবং সেট করা যায়, যেখানে সব child node অন্তর্ভুক্ত থাকে কিন্তু HTML Tags বাদ যায়। এটি CSS স্টাইলিং এর দিকে লক্ষ রাখে না, তাই যদি লুকানো Text থাকে তাহলে তাও দেখায়।

Question-3: What is event delegation in the DOM?

Event delegation হল DOM-এর একটি উপাই যেখানে প্রতিটি child element এ আলাদা আলাদা ইভেন্ট লিসেনার যোগ করার পরিবর্তে, একটি প্যারেন্ট এলিমেন্টে একটি event listener যোগ করা হয়। এই parent element তার child element থেকে bubble up হওয়া ইভেন্টগুলো শুনে। এটি ডাইনামিকভাবে যোগ করা এলিমেন্ট বা অনেকগুলো একই ধরনের এলিমেন্ট এর জন্য খুবই উপকারী।

Question-4: What is event bubbling in the DOM?

Event bubbling is an aspect of the DOM whereby an event that has been fired on a child element "bubbles up" through its parents in the DOM tree. In other words, if you click on a child element, the event will first be handled by the child, then the parent, then the grandparent, and so on, all the way to the document root (usually the window object).

Question-5: How do you create, add, and remove elements using JavaScript?

1. createElement() - For Create Elements

2. appendChild() or insertBefore() - For Add Elements

3. remove() or removeChild() - For Remove Elements

Example -
const div = document.createElement('div') creates a new div
parent.appendChild(div) add it
and div.remove() removes it.
